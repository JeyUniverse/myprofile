## Step-by-Step Process for ZPPSU-Appointment System

1.User Registration

* First-time users must create an account by providing their name, student/staff ID number, email, and password.


* After registration, users log in using their credentials.

2. Dashboard Access

* Upon login, the user is directed to the appointment dashboard.

* The dashboard displays available appointment slots, clinic hours, and upcoming appointments.

3. Appointment Booking

* Users click the “Book Appointment” button.

* They select a date and time from the available slots.

* Users provide any necessary details (e.g., reason for appointment or special instructions).

4. Confirmation

* Once a slot is selected, the user clicks “Confirm Appointment.”

* A confirmation message appears on-screen, and a confirmation email or SMS is sent.

5. Clinic Notification

* The clinic staff is automatically notified of new appointments through their own dashboard.

* They can view the appointment details and prepare accordingly.
